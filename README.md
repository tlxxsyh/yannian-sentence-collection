## 软件说明

### 软件定位

一款优美句子收藏摘录软件，使用户可以持久存储自己喜欢的句子，并支持多种搜索模式。

### 软件功能

1. **本地化、持久化存储软件**：本软件的数据库存储在电脑本地，没有任何服务器万一爆炸了或者电脑连不上网了的风险。并且电子数据比纸质数据更易于搜索和修改，这是本软件强于纸质摘抄的优势所在。
2. **为句子打上标签**：我认为，一个句子同时可能包含多个特征，比如说与友人分别的诗句，那么它可以同时被打上“分别”、“友情”、“伤感”等多个标签，这样当我们寻找句子时也可以根据标签搜索，而不是必须想起那个句子的信息才能去找到那个句子。
3. **支持多种搜索方式**：本软件不止支持通过句子全文搜索，也可以通过句子的作者、国籍、出处、标签、类型、朝代单项搜索或者组合搜索。
4. **直观显示句子库结构**：本软件不仅可以按降序顺序展示每种句子的数量，还可以通过饼图的形式展现句子库的组成结构。
5. **美观**：本软件的 UI 都是经过精心设计，并且征求了多份意见，简洁美观。
6. **易于使用**：例如输入标签时，可以直接采用回车生成标签。从句子漫游页可以直接复制和编辑句子等。
7. **便于迁移**：本软件支持导入和导出句子数据，不会产生离开了本软件句子数据就会丢失的忧虑。

## 导入导出帮助

### 一、文件格式

文件格式：后缀名为. csv 的文件
文件编码：UTF-8

### 二、填写 CSV 的注意事项

为了保证数据能被软件正确识别，填写时请遵循以下规则：

#### 1. 必须保留英文表头

软件是根据第一行的英文单词来识别哪一列是作者，哪一列是内容的。 **第一行必须是以下内容（顺序不限，但拼写不能错）：** `Content,Author,Type,Source,Dynasty,Nationality,Tags,Note,Translation,CreatedAt`

- **不要**把第一行改成中文（比如不要改成“内容,作者...”），否则软件读不出数据。

#### 2. 标点符号的处理

CSV 的原理是用逗号 `,` 来分隔列。

- **如果不使用 Excel，而是直接用文本编辑器写：** 如果你的句子内容里本身包含了**英文逗号** `,`，那么必须用英文双引号 `"` 把整句话包起来。
    - _错误示范：_ `Hello, world, 佚名` (软件会以为 "world" 是作者)
    - _正确示范：_ `"Hello, world", 佚名`
- **如果使用 Excel 编辑：** 你不需要担心这个问题，正常输入即可，Excel 保存时会自动帮你处理引号。

#### 3. 标签 (Tags) 的格式

如果你有多个标签，请使用**中文逗号** `，` 或者 **英文逗号** `,` 分隔。
- 例如：`爱情，友情` 或 `爱情,友情` 都可以。

#### 4. 时间格式 (CreatedAt)

如果你不填写这一列，软件会自动填入当前时间。如果你要填写，建议使用标准格式：
- `2023-10-01 12:00:00`
- 或者 `2023/10/01`

### 三、 标准模版示例

你可以复制下面的内容到记事本，保存为 `template.csv` (编码选 UTF-8)，然后拿去测试导入：

代码段

```
Content,Author,Type,Source,Dynasty,Nationality,Tags,Note,Translation,CreatedAt
"人生若只如初见，何事秋风悲画扇。",纳兰性德,诗词,木兰花·拟古决绝词柬友,清代,中国,爱情,很喜欢的一句,If time could stop at the moment we first met.,2023-01-01 10:00:00
To be, or not to be, that is the question.,莎士比亚,台词,哈姆雷特,,英国,哲学，经典,,生存还是毁灭，这是一个问题
```

### 四、为什么导入文件后会乱码

可能是你的 csv 文件编码错误，有以下两个办法解决：

#### 方法 1：使用“记事本”转换（最简单）

1. 找到你填好的那个 `.csv` 文件。
2. 右键点击文件 -> 选择 **“打开方式”** -> **“记事本”**。
3. 在记事本中，点击左上角的 **“文件”** -> **“另存为”**。
4. 在弹出的对话框底部，找到 **“编码”** 选项。
5. 将其改为 **`UTF-8`** 或 **`UTF-8 带有 BOM`**。
6. 点击保存，覆盖原文件。
7. 再次尝试导入，乱码应该就消失了。

#### 方法 2：使用 Excel 保存（注意选项）

如果你正在使用 Excel 编辑：
1. 点击 **“文件”** -> **“另存为”**。
2. 在保存类型下拉菜单中，**不要**选普通的“CSV (逗号分隔)(*.csv)”。
3. **一定要选** **`CSV UTF-8 (逗号分隔) (*.csv)`**。
4. 保存即可。

## 关于作者

### 为什么要制作这款软件？

其实从高中时就想写这个软件。
一直以来，我看到优美的句子就停不下脚步：看到优美的古诗词，想记下来；看到精彩的比喻句，想记下来；看到别人写的特别生动真切，还是想保存下来以后继续看……不知不觉，各个 app 里的收藏夹里就堆满了一堆句子。但是当我真的想用的时候，我总不能再一个个从这些 app 里找吧？
所以我就想要一个句子管理软件，帮我收藏这些句子。
首先我有几个重要的需求：第一，我需要让所有的句子能在一个地方存储，而不需要我去东翻一个 app，西翻一个 app。第二，我需要给每个句子打上标签，来标注这个句子是关于什么的，又或者这个句子表达的是什么感情。这样当我搜索时，比如说我搜“开心”，那么就会蹦出来一堆开心的句子，当我搜“友情”，那就会蹦出来一堆关于友情的句子。而不是我必须想到原文才能去搜出来这些句子。第三，我希望这些句子可以聚合在一起，我可以清晰看到我的句子库的组织结构。比如说句子库中苏东坡有多少句，中文句子有多少句，英文句子有多少句，各个世界名著的句子又分别有多少句……这样我可以清晰的看出来哪个作家的句子我收藏的最多，哪本书的句子我收藏的更多。
为此我尝试了很多方案：手机备忘录吧，存储量太小，搜也不好搜；Excel 吧，界面的交互太差，而且界面不美观；Obsidian 等笔记软件呢，也是无法满足我的需求。我很惊奇的发现，市面上好像并没有这样一款能实现我的需求的软件，那只好自己写。
于是，这款软件诞生了。
